# 数合美官网管理系统
基于 HTML + TailwindCSS 构建的轻量级官网管理系统，支持案例管理、图片上传、数据导出等功能。

## 功能特点
- 后台登录验证（账号：japhy5000，密码：888888）
- 案例管理：添加/删除案例，支持图片上传（Base64 格式）
- 公司信息编辑：修改核心数据、介绍文案等
- 数据导出：一键导出 cases.json 和 company.json
- 响应式设计：适配电脑/平板/手机等设备

## 使用步骤
1. 访问 index.html 进入官网首页
2. 点击右上角「管理后台」按钮登录
3. 登录后进入 admin.html 管理后台
4. 编辑案例/公司信息，导出 JSON 文件
5. 替换项目中的对应 JSON 文件，完成更新

## 部署说明
1. 将所有文件上传到 GitHub 仓库
2. 开启 GitHub Pages 功能
3. 访问 Pages 地址即可使用
